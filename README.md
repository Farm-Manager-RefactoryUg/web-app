# web-app
Welcome to Tele-Farmer's front-end application using React and Material UI.

## Features
- Log in and Sign up with form validation.
- Password recovery.
- Dashboard.
- Create/Update/Delete a dashboard.
- Edit profile.
- 404 error page.
